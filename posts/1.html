二、游戏作品

点击游戏名即可跳转对应视频链接。

1 . Your Chair

亮点：

  1. 使用 Unity 与 ToLua 框架开发，纯 Lua 代码实现逻辑，兼容 DOTween 动画框架与 UI 系统。
  2. 面向多平台发布。（目前测试了 Windows 和 Andorid，一切正常）

2. 潮汐（DOTS）

‼️: 由于 Unity 在五月底的时候发布了 ecs 1.0 正式版，加上我想将其搬上移动端，项目目前存在一些兼容问题，正在返工修复中……
亮点：

  1. 使用 Unity DOTS 框架开发，性能表现十分优秀。
  2. 本人完全独立开发的第一款完整流程的游戏，拥有别出心裁的关卡与战斗体验，拥有酣畅淋漓的完整游戏体验。

3 . Cell Groth(Compute Shader)

亮点：
使用 Unity Compute Shader 实现视觉渲染与交互逻辑

4. Meet Whale (HDRP)

亮点：

  1. 使用 Unity HDRP 管线进行高质量的视觉渲染。
  2. 使用 VFX 制作了大量的高质量特效
  3. Unity 与 Arduino 连接从而产生别出心裁的游戏交互体验。

5. StarWars（3D射击游戏）

亮点：

  1. 开发了高度可定制化的地图编辑器。
  2. 流畅优雅的俯视角射击游戏体验。
  3. 告别战斗卡顿！优雅的敌人寻路系统与障碍物生成系统（洪水填充算法）
  4. 多样化的武器系统。
  5. 极致优秀的性能表现。

2. Tide（DOTS）

‼️: Due to the release of ecs 1.0 by Unity at the end of May, and my desire to bring it to mobile platforms, there are currently some compatibility issues with the project that are being reworked and fixed…
Highlights:

  1. Developed using the Unity DOTS framework, with excellent performance.
  2. My first independently developed game with a complete process, featuring unique levels and combat experiences, and providing a satisfying overall gaming experience.
